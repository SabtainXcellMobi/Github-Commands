# Github-Commands</br>
This is the github command center where you find out the different github related commands</br></br>


# Create a new repository on the command line: </br></br>

echo "# Github-Commands" >> README.md</br>
git init</br>
git add README.md</br>
git commit -m "first commit"</br>
git branch -M main</br>
git remote add origin https://github.com/SabtainXcellMobi/Github-Commands.git (Your own repo url)</br>
git push -u origin main</br>
</br></br>

# Push an existing repository from the command line:</br></br>


git remote add origin https://github.com/SabtainXcellMobi/Github-Commands.git  (Your own repo url)</br>
git branch -M main  (Your current branch)</br>
git push -u origin main</br>
