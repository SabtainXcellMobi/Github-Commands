# Github-Commands
This is the github command center where you find out the different github related commands


# Create a new repository on the command line: 

echo "# Github-Commands" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/SabtainXcellMobi/Github-Commands.git (Your own repo url)
git push -u origin main


# Push an existing repository from the command line:


git remote add origin https://github.com/SabtainXcellMobi/Github-Commands.git  (Your own repo url)
git branch -M main  (Your current branch)
git push -u origin main
